### Le Eventbrite Like !

Le projet du jour consiste au fait que tu puisse;

  1. Te register (tu recevra un mail).
  
  2. Créer un event.
  
  3. Rejoindre un event (un mail sera envoyé à celui qui a crée l'event).
  
__komment sa marche ?__

~~~~~~~~~~~~~~~~~
$ bundle install

$ rails db:create

$ rails db:migrate

$ rails db:seed1

$ rails console
